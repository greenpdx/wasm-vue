# WASM-Vue

## Install 
@vue/cli https://cli.vuejs.org/<br>
Rustup https://rustup.rs/<br>
wasm-pack https://rustwasm.github.io/wasm-pack/installer/<br>

## Project setup
```
npm install
cargo build
wasm-pack build
```

### Compiles and hot-reloads for development
```
npm run serve
```
